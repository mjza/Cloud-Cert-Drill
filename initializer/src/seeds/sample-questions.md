# Sample AWS Questions

## Question 1

# What is the maximum size of an S3 object?

- [ ] 1 GB
- [ ] 2 GB
- [x] 5 TB
- [ ] 10 TB

**Explanation:**
The maximum size of a single object in Amazon S3 is 5 terabytes. If you want to upload files larger than 5 GB, you must use multipart upload.

**Topic:** S3
**Difficulty:** easy

---

## Question 2

# Which of the following are features of Amazon RDS? (Select 2)

- [x] Automated backups
- [x] Multi-AZ deployment
- [ ] NoSQL database engine
- [ ] Serverless execution

**Explanation:**
Amazon RDS provides automated backups and Multi-AZ (Availability Zone) deployment for high availability. RDS is a relational database service and does not support NoSQL. For serverless options, you would use Amazon DynamoDB or Aurora Serverless.

**Topic:** RDS
**Difficulty:** medium

## Question 3

# What does VPC stand for?

- [ ] Virtual Private Container
- [x] Virtual Private Cloud
- [ ] Virtual Private Cluster
- [ ] Virtual Public Cloud

**Explanation:**
VPC stands for Virtual Private Cloud. It is a logically isolated network environment within AWS that you can launch AWS resources into.

**Topic:** VPC
**Difficulty:** easy
